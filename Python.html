  • Manipulasi String & List

  • File Handling
  • Modular Programming
# Membaca Data dari File
try:
    with open("data.txt", "r") as file:
        isi = file.read()
        print("Isi file:", isi)
except FileNotFoundError:
    print("File tidak ditemukan!")
Semester 3–4

Menuju Aplikasi Nyata: Kekuatan Objek

Ini adalah titik balik di mana saya tidak lagi hanya membuat skrip kecil, melainkan mulai membangun sistem. Dengan konsep Object-Oriented Programming (OOP), saya belajar cara mengorganisir kode seperti di dunia nyata—menggunakan Class dan Object. Saya juga mulai menyentuh dunia antarmuka visual (GUI) dan algoritma yang lebih serius seperti pencarian dan pengurutan data.

  • OOP (Class & Object)
  • Inheritance
  • GUI (Tkinter)
  • Searching & Sorting
  • Stack & Queue
class Mahasiswa:
    def __init__(self, nama, nim):
        self.nama = nama
        self.nim = nim

    def tampilkan_info(self):
        print(f"Mahasiswa: {self.nama} ({self.nim})")

mhs = Mahasiswa("Bagus", "24050974004")
mhs.tampilkan_info()
