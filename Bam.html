PANDUAN PENGGUNAAN

BAM TEKNIK

LANGKAH-LANGKAH PENGGUNAAN (USER)

1. Halaman Awal Aplikasi

  • Menampilkan halaman pembuka dengan tombol “Ayo Belanja Sekarang”.
  • Saat tombol ditekan, pengguna diarahkan ke halaman Katalog Produk.

2. Katalog Produk

  • Menampilkan daftar produk dari database.
  • Setiap produk memiliki kotak centang di samping nama item.
  • Pengguna dapat memilih produk untuk dimasukkan ke keranjang belanja.
  • Setelah memilih produk, pengguna dapat menekan:
  • Checkout → menuju halaman keranjang.
  • Kembali → kembali ke halaman awal.

3. Halaman Keranjang

  • Menampilkan produk yang telah dipilih sebelumnya.
  • Informasi yang ditampilkan:
    • Nama Produk
    • Harga Produk
    • Jumlah Item
    • Stok Tersedia
  • Sistem otomatis menghitung subtotal setiap produk dan total belanja.
  • Pengguna dapat memilih:
    • Kembali → kembali ke katalog.
    • Bayar → menuju halaman pembayaran.

4. Halaman Pembayaran

  • Menyediakan dua metode pembayaran:
    • Cash
    • QRIS
  • Pengguna memilih metode pembayaran lalu menekan tombol Bayar.
  • Sistem akan memvalidasi pembayaran dan menyimpan data transaksi ke database.
  • Jika pembayaran berhasil, pengguna diarahkan ke halaman Terima Kasih.

5. Halaman Terima Kasih

  • Menampilkan pesan:

    • “Terima kasih telah berbelanja bersama BAM Teknik”

  • Tersedia dua tombol:
    • Kembali → menuju halaman awal.
    • Cetak Struk → melihat atau mencetak bukti transaksi.

6. Cetak Struk

  • Sistem mengambil data transaksi terakhir dari database.
  • Struk menampilkan informasi:
    • Daftar produk
    • Jumlah item
    • Total belanja
    • Metode pembayaran
    • Tanggal dan waktu transaksi
  • Struk dapat dicetak atau disimpan sebagai bukti pembelian.

PENGGUNAAN ADMIN

1. Tampilan Awal

  • Pada halaman awal terdapat ikon admin di pojok kiri bawah.
  • Jika ikon diklik, pengguna akan diarahkan ke halaman Login Admin.

2. Halaman Login Admin

  • Admin harus memasukkan:
    • Username : admin
    • Password : admin123
  • Setelah data dimasukkan, tekan tombol Login.
  • Jika berhasil, muncul notifikasi "Login Berhasil".

3. Dashboard Admin

  • Setelah login, admin masuk ke halaman dashboard.
  • Dashboard menampilkan:
    • Total jumlah produk
    • Riwayat transaksi
  • Admin dapat melakukan:
    • Kelola Produk → menambah produk baru.
    • Riwayat → melihat riwayat transaksi.

4. Halaman Manajemen Produk

  • Admin dapat menambahkan produk baru dengan mengisi:
    • Nama Produk
    • Harga Produk
    • Jumlah Stok
    • Upload Gambar Produk
  • Setelah data lengkap, tekan tombol Tambah Produk.
  • Produk akan otomatis muncul pada halaman katalog.

5. Halaman Riwayat Transaksi

  • Menampilkan daftar transaksi pembelian.
  • Admin dapat:
    • Reset → menghapus riwayat transaksi.
    • Kembali → kembali ke dashboard.
