Jejak Langkah Pemrograman C++

Semester 1–2

Gerbang Logika: Fondasi & Sintaks Dasar

Perjalanan C++ saya dimulai dengan memahami struktur program yang lebih disiplin. Jika Python terasa seperti bahasa manusia, C++ memberikan saya pemahaman lebih dalam tentang bagaimana sebuah program dibangun dari nol. Saya belajar tentang `iostream`, cara menangani input-output dengan `cin` dan `cout`, hingga menguasai kontrol alur melalui percabangan dan perulangan.

  • Struktur Dasar (#include)
  • Input/Output (cin/cout)
  • Variabel & Tipe Data
  • Operator Logika
  • Percabangan (if/switch)
  • Perulangan (for/while)
#include <iostream>
using namespace std;

int main() {
    string nama;
    cout << "Masukkan nama: ";
    cin >> nama;
    cout << "Halo " << nama;
    return 0;
}
Semester 2–3

Modularitas: Mengorganisir Kode dengan Fungsi

Memasuki fase menengah, saya mulai belajar cara memecah program besar menjadi bagian-bagian kecil yang lebih mudah dikelola menggunakan fungsi. Di sini saya juga berkenalan dengan Array untuk menyimpan sekumpulan data, serta mulai menyentuh konsep Pointer yang legendaris di C++. Kemampuan untuk membuat kode yang modular membuat program saya jauh lebih efisien.

  • Array 1D & 2D
  • String (C-Style)
  • Function & Parameter
  • Pointer Dasar
  • Rekursi
  • Modular Programming
int tambah(int a, int b) {

Semester 3–4

Dunia Tingkat Lanjut: Pointer, OOP, & Struktur Data

Ini adalah fase yang paling menantang sekaligus memuaskan. Saya menyelam lebih dalam ke konsep Pointer—kekuatan utama C++ yang memungkinkan manipulasi memori secara langsung. Saya juga mulai menerapkan Pemrograman Berorientasi Objek (OOP) untuk membangun sistem yang kompleks, serta mempelajari struktur data seperti Linked List, Stack, dan Queue untuk pengelolaan data yang lebih cerdas.

  • Pointer Lanjutan (*p, &x)
  • Struct & Class
  • Encapsulation & Inheritance
  • Linked List, Stack, Queue
  • File Handling (fstream)
  • Sorting & Searching
// Contoh Pointer & Class
class Mahasiswa {
public:
    string nama;
    void tampil() {
        cout << "Nama: " << nama;
    }
};

int x = 10;
int *p = &x; // p menyimpan alamat x
Analisis & Perbandingan

C++ vs Python: Dua Sisi Mata Uang

Bagi saya, memahami C++ setelah Python (atau sebaliknya) memberikan perspektif yang luar biasa. Python memberikan kemudahan dan kecepatan, sementara C++ memberikan detail dan kontrol penuh. Keduanya sangat penting untuk menjadi seorang pengembang yang komplit.

Fitur Python C++
Tingkat Kesulitan Mudah / High-level Lebih Teknis / Mid-level
Manajemen Memori Otomatis Manual (Pointer)
