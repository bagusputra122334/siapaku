Semester 1–2

Awal Keajaiban: Fondasi & Sintaks Java

Perjalanan Java saya dimulai dengan memahami bahasa yang sangat terstruktur. Berbeda dengan bahasa lain, Java memaksa saya untuk berpikir secara objektif sejak awal. Saya belajar tentang `public class`, metode `main`, serta bagaimana berinteraksi dengan pengguna menggunakan kelas `Scanner`. Di sini, logika pemrograman saya mulai terbentuk dengan disiplin tinggi.

  • Struktur Program Java
  • Input Output (Scanner)
  • Variabel & Tipe Data
  • Operator & Ekspresi
  • Percabangan (if/switch)
  • Perulangan (for/while)
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        System.out.print("Masukkan nama: ");
        String nama = input.nextLine();
        System.out.println("Halo " + nama);
    }
}

Semester 3–4

Mastering OOP: Jantung dari Pemrograman Java

Ini adalah puncak dari pembelajaran Java, di mana konsep Pemrograman Berorientasi Objek (OOP) dipelajari secara mendalam. Saya belajar bagaimana membangun sistem yang aman dengan `Encapsulation`, mewariskan sifat dengan `Inheritance`, hingga fleksibilitas kode melalui `Polymorphism`. Pemahaman tentang `Exception Handling` juga memastikan aplikasi yang saya buat tangguh terhadap kesalahan.

  • Constructor & 'this'
  • Encapsulation (Private)
  • Inheritance (extends)
  • Polymorphism (Overriding)
  • Exception Handling
  • File Handling (FileWriter)
// Contoh Inheritance & Polymorphism
class Hewan {
    void suara() { System.out.println("Hewan bersuara"); }
}

class Kucing extends Hewan {
    @Override
    void suara() { System.out.println("Meong"); }
}
Analisis & Perbandingan

Java vs C++: Perspektif Keamanan & Struktur

Membandingkan Java dengan C++ memberikan saya wawasan tentang manajemen memori. Java menawarkan keamanan melalui JVM dan tidak adanya pointer manual, menjadikannya sangat terstruktur dan "rapi" untuk pengembangan perangkat lunak skala besar.

Fitur C++ Java
Pointer Ada (Manual) Tidak Ada (Aman)
Manajemen Memori Manual / Raw Otomatis (GC)
Struktur Kode Lebih Bebas Sangat Terstruktur
Keamanan Kompleks Lebih Terjamin
